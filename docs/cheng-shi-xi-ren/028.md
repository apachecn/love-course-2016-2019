# 再次驳斥善良或正直的好男人的一个意淫

> 原文：<https://zhuanlan.zhihu.com/p/341512144>

[](https://www.zhihu.com/question/394219918/answer/1625716905)

这位匿名用户的答案提到：

**“采用猎人的世界观方法论，会缩小个体的幸福结果区间，**举个例子我们把幸福的程度量化为（-1，1）吧。如果我们完全地信赖所有人，我们其实是把自己的一切托付给他们，我们的一切仅仅取决于对方的道德，这十分的危险，如果我们遇到了最残忍伪装的人，我们的幸福会被完全破坏，但我们也可能遇到最善良纯洁的人，我们的幸福程度也会很高。很多时候就看我们的运气了。但是采用他的世界观方法论，一开始就立好界限，互相托付，一开始这么做，很明显坦诚不信任对方，这样子当然无法达到最好的幸福模式——互相无条件信赖、互相扶持。但是问问自己，我们自己真的值得信赖么？我们在没有监督的时候，能完全地自我控制么？在婚恋关系中，更忠诚的一方被不那么忠诚的一方辜负简直是司空见惯，只需要查查各种离婚案件的事例就好了。在他推荐的这种模式下，不会达到最好的幸福程度，夫妻之间并不会绝对信任，但是及时止损的话一般不会产生大的危机，至少不会降到负数。原本的幸福区间（-1，1），之后应该就是（-0.2，0.8）了。这个权界契约是相互的，你自己也受其制约。”

————————————————————————

**这位答主的观点是很多好男人的意淫——坏男人得到了“肉”但是得不到“爱”，我现在克制自己虽然得不到“肉”但是未来会得到“爱”。**

我认为真相很可能是这样的，分两点阐述：

一、对于运气并非极好而且硬实力相当的男人。在两性领域很克制保守，恋爱婚姻幸福区间是（-1，1）；在两性领域采纳我鼓吹的世界观方法论，恋爱婚姻幸福区间是（-0.2，1.2）。

理由很简单，经验丰富有助于识别什么样的女人是好女人，有助于识别女性的何种表现更接近真的爱男方，有助于让自己更会与女性相处。

太多老司机给我发私信说他的未婚妻或者妻子**”特别好“。**

二、女性当中有千分之一到万分之一同时具备四个特征：

1、她自己特别好

2、她识人眼光特别毒，总是能够不太困难识别出身边的男性谁人品更好、谁精神境界更高，而且喜欢这一类男人。

3、她物欲淡、不贪图安逸

4、不太看重颜值

好男人在两性领域胜出的唯一的亮光就是不“人性化”的男人与“人性化”的男人同时遇到这种女性，然后该女士一眼就挑出了那个最善良或最正直的男性。这种好事确实存在，但是发生概率极其低，我恳切地希望大家把这种好事忽略不计，类似于你十八岁写人生规划不能有“我会在二十八岁中一笔大奖。”
