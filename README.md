## 下载

### Docker

```
docker pull apachecn0/male-wakeup-wiki
docker run -tid -p <port>:80 apachecn0/male-wakeup-wiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install male-wakeup-wiki
male-wakeup-wiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g male-wakeup-wiki
male-wakeup-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
